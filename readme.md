## Run

#### Create virtual environment

```shell
python3 -m venv ./venv
```

#### Install libraries

- install required libraries

```shell
pip3 install -r requirements.txt
```

- because of some reasons, please installing several libraries manually

```shell
pip install mtcnn
pip install facenet-pytorch
```
