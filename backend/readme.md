## Instruction

- Download `shape_predictor_68_face_landmarks`

```shell
wget http://dlib.net/files/shape_predictor_68_face_landmarks.dat.bz2
bzip2 -d shape_predictor_68_face_landmarks.dat.bz2
```

- Install package

```shell
pip3 install -r requirements.txt
```
